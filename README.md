# 🚀 3D Portfolio Website

A modern and interactive **3D Portfolio Website** built using **HTML, CSS, and JavaScript**.
This project focuses on clean UI, smooth animations, and a modular code structure for scalability and maintainability.

---

## ✨ Features

* 🎨 Modern dark-themed UI with gradients
* 🧊 3D hover effects using CSS transforms
* ⚡ Smooth scrolling and animations
* 🧠 Modular JavaScript architecture
* 🖱️ Custom cursor effects
* 🔝 Back-to-top button
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (ES Modules)
